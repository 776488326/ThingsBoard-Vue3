## thingsboard 前端工程

### 核心技术栈

- vite 打包工具
- vue3 界面框架
- typescript 脚本语言
- scss 样式
- element-plus 组件库
- vue-router 路由
- pinia 状态管理

### 项目内容

- 使用 Vue3 技术栈重新开发 thingsboard 前端界面
